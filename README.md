# 🎈 UnGlobo - Risk It!

Un divertido juego multiplayer donde inflas un globo sin saber cuándo explotará. ¡El que arriesga más gana!

## 🎮 ¿Cómo jugar?

1. **Configura el juego**: Elegí la cantidad de jugadores (2-8) y rondas (1-10)
2. **Nombres personalizados**: Cada jugador ingresa su nombre
3. **Inflar el globo**: Mantené presionada la barra **ESPACIO** para inflar
4. **Cobrar puntos**: Soltá antes de que explote para asegurar tus puntos
5. **Gana quien tenga más puntos** al final de las rondas

### ⚠️ ¡Cuidado!
El globo puede explotar en cualquier momento entre 1.5x y 5x su tamaño original. ¡No sabés cuándo explotará!

## 🎯 Características

- ✅ **Multiplayer** (2-8 jugadores)
- ✅ **Nombres personalizados** para cada jugador
- ✅ **Varias rondas** configurables
- ✅ **Efectos visuales** hermosos (partículas, explosiones, animaciones)
- ✅ **Sistema de puntuación** por tiempo inflado + tamaño del globo
- ✅ **Ranking final** con los ganadores
- ✅ **Compatible con móvil** (táctil)

## 🚀 Cómo ejecutar

### Versión Web (Navegador)

Simplemente abrir `index.html` en cualquier navegador moderno.
